Technical Overview

What's Built In

01
Design & Frontend

Premium dark theme with Matte Black, Gold, and Dark Grey palette. Bebas Neue headlines, Montserrat body. Fully responsive grid across mobile, tablet, and desktop. Zero external framework dependency — pure HTML/CSS/JS.

HTML5 CSS3 Vanilla JS Bebas Neue Montserrat Mobile First
02
Payment Integration

Razorpay handles all Indian transactions — UPI, debit/credit cards, net banking, and COD. PayPal handles international orders. Stripe integration planned for Phase 2. Secure encrypted checkout throughout.

Razorpay ✓ PayPal ✓ UPI COD India Stripe (Phase 2)
03
Shipping & Logistics

Shiprocket integration for Pan-India delivery. Automatic tracking updates via SMS and email. Country-specific delivery timelines displayed automatically. Premium protective packaging on all orders.

Shiprocket ✓ Pan India 7 Countries Live Tracking
04
Multi-Currency & Geo-Pricing

Automatic visitor country detection. Prices display in local currency — INR, USD, AED, GBP, EUR, AUD. Admin can configure per-country pricing, shipping charges, and tax rules from backend without changing code.

INR · USD · AED GBP · EUR · AUD Geo-Detection Admin Editable
05
SEO & Performance

Custom meta titles and descriptions per page. Schema markup support. Clean semantic HTML structure. Fast loading with optimised image delivery via CDN. Cookie policy and privacy-compliant for Meta Ads and Google Ads.

SEO Ready Schema Markup CDN Images Cookie Policy Meta Ads Ready
